# @codecanon/next-presets

Allow your user to choose from a list of shadcn presets.

## Installation

```bash
bun add @codecanon/next-presets
```

```css
/* Then add to your global css file */
@import "@codecanon/next-presets/styles.css";
```

## Development

```bash
bun install
bun run build        # compile library
bun run dev          # watch mode
bun test             # run tests
bun run typecheck    # type-check core + react entry
bun run format       # format with prettier
bun run play         # start the React playground (playground/)
```
